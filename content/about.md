# About this project

This is a content reference wiki for **OAM 330: Principles of Organization & Management** (Prof. Williams, Spring 2026), built by a student in the course and made open-source so it can be reused or adapted for future cohorts. It is not an official course resource and is not affiliated with the College.

## What's in it

- **132 concept pages** — one short page per term taught in the course, organized by module.
- **22 reading summaries** — the main argument, key terms, and examples from each assigned reading. Source PDFs and slide decks are not redistributed.
- **12 module pages** — a one-page overview per module: what the module is about, the core empirical findings, and the concept pages it covers.
- **5 cross-cut pages** — synthesis pages that span modules (cultural differences, fairness/justice, System 1 vs. System 2, decision-making biases, distractor pairs).

## How it was built

I built this wiki throughout the semester as I worked through the course. It pulls together lecture slides, assigned readings, and the study guides Williams distributed, and condenses each idea into a short, linkable page. Slides set the canonical phrasing; where readings extend or disagree, the reading's view is noted alongside.

The structure is loosely adapted from [Karpathy's LLM Wiki](https://github.com/karpathy/LLM101n) pattern: small atomic pages, dense cross-linking, and a flat index instead of nested folders. The drafting was assisted by LLM tooling against my own notes and the course materials.
