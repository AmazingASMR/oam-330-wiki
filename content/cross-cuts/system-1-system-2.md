# Cross-cut — System 1 vs. System 2

## Why this matters
Kahneman's two-system model is the most heavily referenced single framework in OAM 330 — it is named the "master frame" of Module 3 and is invoked again in Module 6 (central/peripheral routes) and Module 11 (ethical fading). Synthesis questions are likely to ask which system is operating in a given scenario or to swap the System 1 / System 2 mapping in distractors.

## The frame
**System 1** = limbic/amygdala, automatic, fast, effortless, **not under conscious control**, parallel — snap judgments, gut reactions, practiced routines. **System 2** = prefrontal cortex, conscious, slow, effortful, requires attention. They run **simultaneously**, not sequentially. Slide quote: *"System 2 doesn't always know when System 1 is operating — we frequently think we are making a more rational, data-driven decision than we are."* System 1 isn't the enemy; it's the default. System 2 is effortful, so we conserve it, and biases slip through.

## Module 3 — Decision Making
Every Module 3 cognitive bias is a [[concepts/system-1-system-2|System 1]] process that System 2 fails to catch. Slide buckets:
- Perception biases (System 1 distorts the input): [[concepts/perception-vs-reality]], [[concepts/motivated-perception]], [[concepts/self-fulfilling-prophecies]]
- Evidence-handling biases: [[concepts/availability-heuristic]], [[concepts/confirmation-bias]], [[concepts/halo-effects]]
- Decision biases — ego protection ("we want to think well of ourselves"): [[concepts/overconfidence]], [[concepts/hindsight-bias]]
- Decision biases — "we're not computers": [[concepts/anchoring-bias]], [[concepts/framing-and-risk]], [[concepts/sunk-cost-fallacy]]
- Debiasing: [[concepts/reducing-decision-biases]] — Gilbert: *"the brain cannot see itself fooling itself"*; engaging System 2 via awareness alone is insufficient — **avoid the situations** that produce bias.

## Module 6 — Influence & Persuasion
The central/peripheral distinction maps cleanly onto System 2 / System 1.
- [[concepts/central-vs-peripheral-route]] — central route engages System 2 (argument quality, durable change); peripheral engages System 1 (cues, faster but more fragile). They co-occur.
- All six Cialdini principles are peripheral-route, System 1 cues: [[concepts/cialdini-authority]], [[concepts/cialdini-reciprocity]], [[concepts/cialdini-commitment-consistency]], [[concepts/cialdini-scarcity]], [[concepts/cialdini-liking]], [[concepts/cialdini-social-proof]]

## Module 11 — Ethics
The "should self" vs. "want self" gap is a System 2 / System 1 conflict that plays out across the prediction → decision → recollection arc.
- [[concepts/ethical-fading-visceral]] — decision-stage System 1 mechanism: ethics drops out of awareness (fading) or hot bodily drives override deliberation (visceral).
- [[concepts/ethical-forecasting-errors]] — System 2 forecasts the "should self"; System 1 acts at the moment; mismatch produces over-prediction of ethical behavior.
- [[concepts/ethical-memory-revisionism]] — System 1 reconstructs the memory post-hoc so we still see ourselves as ethical, sustaining the cycle.

## Common points of confusion
- Distractors will reverse the mapping — e.g., claim peripheral route engages System 2 or that Cialdini principles are central-route.
- Distractors will claim the two systems run **sequentially** — they run **simultaneously**.
- "Slow down / be aware" looks like the obvious debiasing answer, but slides explicitly say awareness alone isn't enough (Gilbert) — structural fixes beat willpower.
- Ethical fading is **not** moral justification — fading = System 1 not seeing the ethical issue at decision time; justification = post-hoc rationalizing story. Memory revisionism is the *recollection-stage* sibling.
- Dunning-Kruger direction: **less** skill → **more** overconfidence (counterintuitive).
- System 1 is the **default**, not "bad" — it gives us reflexes we need; the trouble is when systems conflict.

## Synthesis question patterns
- "A manager hiring a candidate finds her résumé unimpressive after a weak handshake but rates her warm and competent after a firm one. Which system is at work, and which Module 3 bias is this closest to?" (System 1; [[concepts/halo-effects]])
- "A pitch leans on a celebrity endorser and a 'limited-time' offer. Which route, which system?" (Peripheral route, System 1 — [[concepts/cialdini-authority]] + [[concepts/cialdini-scarcity]])
- "An executive crunches the safety numbers as a cost-benefit analysis and never registers the lives-at-stake angle. This is best described as…" (Ethical fading, a System 1 decision-stage failure — distractor: moral justification, which is System 2 post-hoc).
