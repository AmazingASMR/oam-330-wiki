# Cross-cut — Distractor pairs (cram sheet)

The single most useful page for the morning of the exam. Every entry below is a real distractor pair flagged on a concept page's `## Key distinctions` section. Memorize the *distinguishing feature* — that's what flips a 50/50 to a correct answer.

## How to use this page

Read each pair's distinguishing feature aloud. If you can recite it cold, move on. If you hesitate, click the linked concept page and re-read.

## Module 2 — Empirical Approaches

| Pair | Distinguishing feature |
|---|---|
| [[concepts/random-assignment]] vs. random sampling | *Assignment* = how subjects go into conditions (internal validity / causation); *sampling* = how subjects are selected from a population (external validity / generalizability). Most labs have one but not the other. |
| [[concepts/rcts-experiments]] vs. [[concepts/quasi-experiments-correlational]] | RCT = experimenter randomly assigns; quasi = uses *natural/existing* division. Quasi cannot establish causation cleanly. |
| [[concepts/rcts-experiments]] vs. correlational study | RCT manipulates the IV; correlational only *measures* variables as they exist. |
| [[concepts/alternative-explanations-correlation]] — the three alternatives | Spurious / reverse direction / third variable. **Exactly three.** Trap distractors: "measurement error," "selection bias," "chance." |
| [[concepts/n-of-1-problem]] vs. correlation problem | N-of-1 = not *enough* cases; correlation = even with many cases, can't infer causation without manipulation. Two distinct reasons evidence > anecdote. |
| [[concepts/correlation-vs-causation]] | Correlation = measured association; causation = established only via experiment + random assignment. |

## Module 3 — Decision Making

| Pair | Distinguishing feature |
|---|---|
| [[concepts/availability-heuristic]] vs. [[concepts/confirmation-bias]] | Availability = passive *recall* ease (what pops up); confirmation = active *seeking* (what we go look for). |
| [[concepts/anchoring-bias]] vs. [[concepts/framing-and-risk]] | Anchoring shifts a *starting number* (magnitude); framing shifts *risk preference* by gain/loss wording. |
| [[concepts/anchoring-bias]] vs. [[concepts/sunk-cost-fallacy]] | Anchoring = *initial reference number*; sunk cost = *past investment* of money/time/effort. |
| [[concepts/overconfidence]] vs. [[concepts/hindsight-bias]] | Overconfidence = *forward* ("I'll predict right"); hindsight = *backward* ("I knew it all along"). They reinforce each other. |
| [[concepts/halo-effects]] vs. [[concepts/self-fulfilling-prophecies]] | Halo = perceiver-side trait generalization (you misjudge); SFP = target's *actual behavior* changes in response to expectations. |
| [[concepts/halo-effects]] vs. [[concepts/hindsight-bias]] | Halo recolors *traits of the actor* (decisive vs. rigid CEO); hindsight recolors *predictability of the outcome*. |
| [[concepts/halo-effects]] vs. [[concepts/confirmation-bias]] | Halo generalizes across *traits of one target*; confirmation generalizes evidence search toward *a belief about anything*. |
| [[concepts/perception-vs-reality]] vs. [[concepts/motivated-perception]] | Perception/reality = general claim (we don't see reality directly); motivated = the specific case where *desires* shape perception. |
| [[concepts/system-1-system-2]] | System 1 = fast, automatic, limbic; System 2 = slow, effortful, prefrontal. They run *simultaneously*, not sequentially. |
| [[concepts/sunk-cost-fallacy]] vs. [[concepts/framing-and-risk]] | Sunk cost = pull of *already-spent* resources; framing = present option presentation. Lost-ticket vs. lost-cash is the canonical mirror pair. |

## Module 4 — Personality & Culture

| Pair | Distinguishing feature |
|---|---|
| [[concepts/myers-briggs]] vs. [[concepts/ocean-traits]] (Big 5) | MBTI = theory-driven, dichotomous, weak retest (~50% reclassify per Grant); Big 5 = empirical, continuous, modestly predictive. |
| [[concepts/myers-briggs]] vs. [[concepts/barnum-effect]] | MBTI feels accurate largely *because* of the Barnum effect — vague descriptors feel personal. |
| [[concepts/individualism-collectivism]] vs. [[concepts/power-distance]] | Different Hofstede dimensions: I/C = group orientation; PD = acceptance of *hierarchy*. Don't conflate. |
| Individualism vs. "selfish" | Slides frame it normatively as authenticity/initiative, NOT selfishness. Common distractor. |

## Module 5 — Motivation

| Pair | Distinguishing feature |
|---|---|
| [[concepts/expectancy-theory]] vs. [[concepts/equity-theory]] | Expectancy = *forward* effort→reward chain (multiplicative; one zero kills motivation); equity = *sideways* comparison to a referent. |
| [[concepts/distributive-vs-procedural-justice]] | Distributive = outcome fairness; procedural = process fairness. **Procedural is the stronger predictor of job performance** (slide-flagged counterintuitive). |
| [[concepts/equity-theory]] vs. [[concepts/distributive-vs-procedural-justice]] | Equity = the *mechanism* (input/output ratio comparison); distributive justice = the *feeling* of outcome fairness equity produces. |
| [[concepts/cultural-differences-fairness]] | Three allocation principles: **merit / equality / need**. Equity = individualist default; *desire for fair process* is universal. |
| [[concepts/intrinsic-vs-extrinsic-motivation]] | Only **contingent/expected** rewards corrupt intrinsic motivation; unexpected/non-contingent ones don't (Lepper-Greene preschoolers). |
| [[concepts/flow-state]] 2×2 | Flow = high skill + high challenge. Apathy = low+low; boredom = high skill + low challenge; anxiety = low skill + high challenge. |
| [[concepts/flow-state]] vs. job satisfaction | Flow = moment-to-moment state during a task; satisfaction = global evaluation. |
| [[concepts/job-satisfaction-performance]] | Correlation **~.30 ("medium")** — not zero, not strong. Distractor: "no relationship" or "very strong." |

## Module 6 — Influence & Persuasion

| Pair | Distinguishing feature |
|---|---|
| [[concepts/central-vs-peripheral-route]] | Central = System 2 argument quality, durable; peripheral = System 1 cues, fragile. They co-occur, not mutually exclusive. |
| [[concepts/cialdini-scarcity]] vs. [[concepts/cialdini-social-proof]] | Both spike urgency, but: scarcity = "*few* can have it" (exclusivity); social proof = "*many* are choosing it" (consensus). Opposite signals. |
| [[concepts/cialdini-authority]] vs. [[concepts/cialdini-social-proof]] | Authority = *vertical* (defer to expert above); social proof = *horizontal* (follow similar peers). |
| [[concepts/cialdini-liking]] vs. [[concepts/cialdini-social-proof]] | Liking = *one-to-one* affective bond; social proof = *many-to-one* peer behavior. |
| [[concepts/cialdini-reciprocity]] vs. [[concepts/cialdini-commitment-consistency]] | Reciprocity binds you to repay *another's* gift; consistency binds you to *your own* prior statement. |
| [[concepts/cialdini-commitment-consistency]] ladder | Active + public + voluntary > active + public > active > passive default. **Coerced commitments backfire.** |
| [[concepts/asch-conformity]] vs. Milgram (obedience) | Asch = conformity to *peers* (74% conformed at least once); Milgram = compliance with *authority* (2/3 to "XXX"). |
| [[concepts/order-effects]] | Recency wins in *serial-judgment competitions* ("the last shall be first" — figure skating, *American Idol*). Primacy wins after delay. |

## Module 7 — Power & Leadership

| Pair | Distinguishing feature |
|---|---|
| [[concepts/power-paradox]] vs. [[concepts/power-changes-behavior]] | Paradox = the *circular self-defeating loop* (capacities that won you power get switched off); changes = the *three-item behavioral list* (action ↑, perspective-taking ↓, pressure of others ↓). |
| [[concepts/power-paradox]] vs. hubris syndrome | Paradox = general psychological pattern (shows up in 10-min priming); hubris syndrome (Owen) = clinical disorder requiring *sustained* power + minimal constraint. |
| [[concepts/power-distance]] vs. [[concepts/power-changes-behavior]] | PD = *cultural/structural*; power-changes = *individual-psychological*. |
| [[concepts/leader-emergence-factors]] vs. leader effectiveness | Emergence = who gets *perceived* as leader; effectiveness = whether they *produce good outcomes*. Emergent ≠ effective. |
| Receiver- vs. transmitter-oriented communication ([[concepts/cultural-power-leadership]]) | High-PD: listener decodes; low-PD: speaker is responsible for clarity. Drives mitigated speech (KAL 801). |

## Module 8 — Social Identity & Diversity

| Pair | Distinguishing feature |
|---|---|
| [[concepts/discrimination-and-fairness]] vs. [[concepts/access-and-legitimacy]] | Paradigm 1 = *moral/legal* / "right thing"; paradigm 2 = *strategic/market* / "profitable thing." Spanish-speaking reps for Latino market = A&L, not D&F. |
| [[concepts/stereotype-threat]] vs. [[concepts/stereotype-tax]] | Threat = target's *own* performance drops under identity salience (internal); tax = career *cost paid* from others' System 1 inference (external). |
| [[concepts/stereotype-threat]] vs. [[concepts/workplace-stigmatization]] | Stigmatization = interpersonal devaluing by others; threat = internal performance hit even with no overt mistreatment. |
| [[concepts/diversity-innovation-relationship]] vs. [[concepts/access-and-legitimacy]] | Better-ideas = *internal* problem-solving from heterogeneity; A&L = *external* market access. Both strategic, opposite directions. |
| [[concepts/why-employees-care-diversity]] | Authentic AND not conspicuous. Distractor: "to comply with anti-discrimination law" (that's the org's reason, not the employee's). |
| [[concepts/social-identity-definition]] vs. demographic diversity | Demographics = observable categories; social identity = *subjective importance* attached to them. |

## Module 9 — Negotiation

| Pair | Distinguishing feature |
|---|---|
| [[concepts/batna]] vs. [[concepts/reservation-price]] | BATNA = the outside *alternative*; reservation = the *number* it produces. Olivia ($31K reservation, accepted $32K = Mode B) and Shapiro ($7.4M EV BATNA, rejected $8.5M = Mode A) are the canonical violations. |
| [[concepts/batna]] vs. target point | Target = highest *aspiration*; BATNA = walk-away alternative. Classic prep mistake to confuse them. |
| [[concepts/batna]] vs. fairness anchor | A "fair" number is NOT a BATNA. Shapiro rejected $8.5M because it felt unfair vs. $10M, but his EV BATNA was $7.4M — should have accepted. |
| [[concepts/distributive-vs-integrative-bargaining]] | Distributive = 1 issue, fixed-pie zero-sum, guarded info, anchor first / anger / deadline; integrative = multiple issues, expand-the-pie, open about goals. |
| [[concepts/positions-vs-interests]] | Position = *what* I ask for; interest = *why*. Compromise splits the position; interest-based work serves both interests in full (orange/cake-and-juice). |

## Module 10 — Groups & Teams

| Pair | Distinguishing feature |
|---|---|
| [[concepts/groupthink-and-groupshift]] | Groupthink = *suppressed dissent → artificial consensus*; groupshift = *amplified majority position → polarization*. Both group failure modes; treat as DISTINCT. |
| [[concepts/groupthink-and-groupshift]] vs. [[concepts/common-knowledge-effect]] | Groupthink = dissent suppression; common-knowledge = info-sampling failure (over-discuss shared, under-discuss unshared). Grogan Air ~75% wrong VP. |
| [[concepts/group-cohesion]] vs. performance norms | Cohesion alone does NOT predict productivity. Productivity = cohesion **×** performance norms (Hawthorne bank-wiring: high cohesion + low norms → low output). |
| [[concepts/social-loafing]] vs. groupthink | Loafing = effort reduction; groupthink = dissent suppression. Both group dysfunctions, mechanistically distinct. |
| [[concepts/social-loafing]] vs. production blocking | Loafing = willful effort reduction; production blocking = cognitive interference from many people talking at once (brainstorming). |
| [[concepts/group-brainstorming]] vs. nominal group technique | Counterintuitive: NGT *outperforms* brainstorming on idea count *and* quality. Production blocking degrades brainstorming. |
| [[concepts/mechanistic-vs-organic]] | Mechanistic = top-down / formalized / narrow span (Wal-Mart, railroads); organic = fluid / distributed / wider span (Apple, startups). Mechanistic ≠ inherently bad. |
| [[concepts/punctuated-equilibrium-model]] | Transition hits at the **midpoint** regardless of total time. Applies only to *temporary* deadline-driven groups. |
| [[concepts/psychological-safety]] vs. [[concepts/group-cohesion]] | Cohesion = bond; psychological safety = willingness to *dissent and speak up*. Direct opposite of groupthink on the speak-up dimension. Project Aristotle. |
| [[concepts/roles-and-norms]] | Role perception (you) vs. role expectations (others). Role conflict (within one role) vs. interrole conflict (across memberships). |

## Module 11 — Ethics

| Pair | Distinguishing feature |
|---|---|
| [[concepts/ethical-fading-visceral]] | Fading = *cognitive* (ethics drops out of view; Pinto cost-benefit); visceral = *bodily/affective* (hot drives override; hunger, fear, anger). Both decision-stage; keep separate per slide. |
| [[concepts/ethical-fading-visceral]] vs. [[concepts/moral-justification]] | Fading = ethics *never enters awareness*; justification = ethics *acknowledged but explained away* (post-hoc). |
| [[concepts/moral-justification]] vs. [[concepts/social-norms-unethical-behavior]] | Justification = *internal rationale* ("I have a reason"; inequity, anonymity, Pythias Brown TSA, Enron); norms = *external/social cue* ("everyone does it"; pluralistic ignorance, Milgram). |
| [[concepts/moral-justification]] vs. [[concepts/outcome-focused-culture]] | Justification = in *one employee's head*; outcome focus = *cultural / leadership-built / incentive-structured* (Pinto, Wells Fargo "Eight is great"). |
| [[concepts/moral-inaction]] vs. [[concepts/moral-justification]] | Justification = rationalize *one's own* act; inaction = fail to confront *someone else's* act. ~2/3 non-reporting rate. |
| [[concepts/moral-inaction]] vs. obedience | Obedience = *active* compliance with directive (Milgram, slide 15 = social norms); inaction = *passive* failure to intervene. |
| [[concepts/ethical-memory-revisionism]] vs. [[concepts/ethical-fading-visceral]] | Fading = ethics invisible *at decision time*; revisionism = ethics *re-read out of memory* afterward ("that wasn't really an ethical issue"). |
| [[concepts/ethical-forecasting-errors]] vs. fading/visceral | Forecasting errors = *before* (predict you'd be ethical); fading + visceral = *at decision time*; revisionism = *after*. |
| [[concepts/ethics-antidote]] vs. [[concepts/outcome-focused-culture]] | Direct opposites. Antidote = process + transparency + checks/balances + dissent (J&J 1982 Tylenol). Outcome focus = ends justify means (Enron). |
| [[concepts/ethics-antidote]] vs. compliance training alone | Antidote is *culture*, not policy/training. Synergistic corruption (Enron) shows formal codes fail without culture. |
| [[concepts/bounded-ethicality]] vs. hypocrisy | Hypocrisy implies *awareness*; bounded ethicality is the *absence* of awareness. |
| [[concepts/want-self-vs-should-self]] | Should = abstract / hypothetical / System 2; want = concrete / live decision / System 1. Sexual-harassment interview: 62-68% predict confronting; 0% actually refuse. |

## Module 12 — Happiness & Wrap-up

| Pair | Distinguishing feature |
|---|---|
| [[concepts/happiness-stability]] — three drivers | Genetics (set point; Minnesota Twins) / adaptation (lottery + marriage adapt within ~2 yr) / social comparison (Vladimir's Choice; Olympics bronze > silver). Distractor: confusing adaptation with social comparison. |
| [[concepts/happiness-improvement]] | Experiential > material (3 reasons: no comparison, hard to adapt, often social); prosocial spending > self-spending; weak ties also help. |
| [[concepts/happiness-vs-objective-quality]] | Easterlin paradox: objective conditions (GDP, plumbing, life expectancy) soared 1940→today; subjective happiness flat. "Money matters at the bottom, decouples at the top" (Kahneman-Deaton). |
| [[concepts/happiness-consequences]] | Happiness as a *cause* of downstream goods, not just a result. Longevity gain from happiness exceeds gain from not smoking. |

## High-yield "exact" study-guide quotes

- **3 alternative explanations for correlation:** spurious / reverse direction / third variable (NOT measurement error / chance / selection bias).
- **3 obstacles to experimentation:** impractical / impossible / unethical.
- **3 mega-takeaways** ([[concepts/mega-takeaways]]): (1) human behavior is predictable; (2) what we think ≠ reality; (3) other people matter.
- **4 enabling mechanisms of organizational misconduct (M11):** [[concepts/moral-justification]] / [[concepts/social-norms-unethical-behavior]] / [[concepts/outcome-focused-culture]] / [[concepts/moral-inaction]].
- **Antidote pillars** ([[concepts/ethics-antidote]]): transparency / checks & balances / process focus / dissent.
- **3 conformity-reducers** ([[concepts/asch-conformity]]): one ally, written answers, small group.
- **5 R&J social-loafing levers:** group goals / inter-group competition / peer evals / select for motivation / reward identifiable individual contributions.
- **7 R&J cohesion levers:** smaller / agreement on goals / time together / status of joining / external competition / group rewards / physical isolation.
- **Pink's three intrinsic drivers:** autonomy / mastery / purpose.
- **Moral Foundations — 5 categories** (verify against slide deck — page is `medium` confidence): see [[concepts/moral-foundations-five]].
- **Asian disease problem (framing):** "200 saved" → 72% safe; "400 die" → 78% gamble. Same outcome, opposite preference.
